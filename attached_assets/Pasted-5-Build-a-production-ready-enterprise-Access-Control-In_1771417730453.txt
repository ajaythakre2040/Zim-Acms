5 Build a production-ready enterprise Access Control Integration System.

PROJECT NAME:
Unified Access Intelligence Platform (UAIP)

OBJECTIVE:
Integrate and normalize data from:
1) ESL eTimeTracker database (Attendance logs)
2) Door Access Controller database (Door entry logs)
3) ZIN Access Control Management System database

The system should unify, analyze, and visualize logs in real-time with rule-based automation.

3) RULE ENGINE (IMPORTANT)

- Zone-based restrictions

- Department-based rules
- Auto-expiry blocking
