PROJECT NAME:
ZIN Unified Access Intelligence Platform (ZUAIP)

OBJECTIVE:
We already have database access credentials for:

1) ESL_DB (eTimeTracker Database)
2) CONTROLLER_DB (Access Controller Database)

The system must directly fetch data from these databases
and build a centralized dashboard system with rule intelligence.

--------------------------------------------------------

We already have DB access.
System must:

- Fetch attendance logs from ESL_DB
- Fetch employee master from ESL_DB
- Fetch access logs from CONTROLLER_DB
- Fetch users from CONTROLLER_DB
- Fetch Access Control Rules from CONTROLLER_DB

No duplicate storage.
Maintain last_synced_timestamp.
